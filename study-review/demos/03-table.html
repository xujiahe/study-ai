滚动鼠标滚轮体验 10 万行虚拟滚动

虚拟滚动原理

无论数据多少行，只绘制可视区域内的行。

起始行：startRow = Math.floor(scrollY / rowHeight)

结束行：endRow = startRow + Math.ceil(viewH / rowHeight) + 1

行 Y 坐标：y = row * rowHeight - scrollY

10 万行数据与 100 行数据的渲染开销几乎相同。

滚动鼠标滚轮，观察冻结列效果

冻结列实现

将表格拆分为 4 个区域，分别绘制：

• 左上角固定区域（不滚动）

• 表头（只随水平滚动）

• 冻结列（只随垂直滚动）

• 数据区域（双向滚动）

坐标公式：x = colOffset - scrollX，y = row * rowH - scrollY

单元格合并

合并单元格宽度：sum(colWidths[col..col+colspan-1])

合并单元格高度：rowspan * rowHeight

被合并的子单元格跳过绘制（用 Set 标记）。

坐标：x = colOffsets[col]，y = row * rowHeight
