变换矩阵

Canvas 变换作用于坐标系，而非图形本身。

translate(x,y) 平移原点

rotate(angle) 绕当前原点旋转（弧度）

scale(sx,sy) 缩放坐标系

绕任意点旋转：先 translate 到旋转中心，rotate，再 translate 回来。

save()/restore() 隔离变换作用域。

像素操作

getImageData(x,y,w,h) 返回 ImageData，.data 是 RGBA 字节数组。

像素索引：idx = (y * width + x) * 4

灰度公式：0.299R + 0.587G + 0.114B（人眼感知权重）

频繁读取时设置 willReadFrequently: true 优化性能。

save / restore / clip

save() 将当前状态（样式 + 变换矩阵 + 裁剪区域）压栈。

restore() 弹出栈顶状态，唯一重置裁剪区域的方式。

clip() 多次调用取交集，裁剪区域只会越来越小。

clip('evenodd') 奇偶规则，可实现镂空效果。
