路径绘制核心 API

beginPath() 每次绘制新图形前必须调用，否则路径会叠加。

moveTo / lineTo 移动画笔 / 画直线。

arc(x,y,r,start,end) 绘制圆弧，角度为弧度制。

closePath() 连回路径起点，形成封闭图形。

扇形 = moveTo 圆心 + arc + closePath。

贝塞尔曲线

quadraticCurveTo(cpx,cpy,x,y) 二次贝塞尔，1 个控制点。

bezierCurveTo(cp1x,cp1y,cp2x,cp2y,x,y) 三次贝塞尔，2 个控制点。

拖动滑块改变波形数量和振幅，观察控制点对曲线的影响。

渐变与阴影

createLinearGradient(x0,y0,x1,y1) 线性渐变，坐标为画布绝对坐标。

createRadialGradient(x0,y0,r0,x1,y1,r1) 径向渐变，两圆圆心偏移可模拟光照。

shadowColor / shadowBlur / shadowOffsetX/Y 阴影属性，绘制完后记得重置为 transparent，避免影响后续绘制性能。

线条样式

lineCap：butt（平直）/ round（圆头）/ square（方头）

lineJoin：miter（尖角）/ round（圆角）/ bevel（斜切）

setLineDash([实线,间隔]) 绘制虚线，lineDashOffset 可做蚂蚁线动画。
