# Waylines.wpml 协议

## 概述

Waylines.wpml 是执飞协议文件，包含无人机实际执行飞行所需的全部信息。它由 Template.kml 经过算法计算生成，包含具体的航点坐标和动作序列。

## XML 结构

```xml
<?xml version="1.0" encoding="UTF-8"?>
<wpml xmlns="http://www.gdu.com/wpmz">
    <version></version>
    <mission>
        <!-- 任务配置 -->
    </mission>
    <folders>
        <!-- 航线文件夹 -->
    </folders>
</wpml>
```

## 字段说明

### WaylinesWpml（根元素）

| 字段名 | 类型 | 描述 |
|--------|------|------|
| version | string | 协议版本号 |
| mission | object | 任务配置 |
| folders | array | 航线文件夹数组 |

### MissionConfig（任务配置）

| 字段名 | 类型 | 描述 | 取值范围 |
|--------|------|------|----------|
| drone | object | 无人机信息 | 详见[无人机类型](/protocols/enums/drone) |
| payloads | array | 载荷信息数组 | 详见[载荷类型](/protocols/enums/payload) |
| flyToWaylineMode | string | 飞向航线模式 | "safely": 安全模式, "directly": 直接模式 |
| finishAction | string | 完成动作 | "goHome": 返航, "noAction": 悬停, "jumpFly": 机库跳飞, "safeGoHome": 安全点返航, "alongGoHome": 沿航线返航 |
| exitOnRCLost | string | 遥控器丢失行为 | "goContinue": 继续, "executeLostAction": 执行失控动作 |
| executeRCLostAction | string | 失控执行动作 | "goBack": 返航, "hover": 悬停 |
| takeOffSecurityHeight | number | 起飞安全高度 | 1-500 米 |
| globalTransitionalSpeed | number | 全局过渡速度 | 1-15 m/s |
| autoReroute | object | 自动绕障配置 | - |

### AutoReroute（自动绕障配置）

| 字段名 | 类型 | 描述 | 取值范围 |
|--------|------|------|----------|
| missionAutoRerouteMode | number | 任务航线绕行模式 | 0: 关闭, 1: 开启 |
| transitionalAutoRerouteMode | number | 过渡航线绕行模式 | 0: 关闭, 1: 开启 |

### Folder（航线文件夹）

| 字段名 | 类型 | 描述 | 取值范围 |
|--------|------|------|----------|
| templateId | number | 模板 ID | - |
| executeHeightMode | string | 执行高度模式 | "relativeToStartPoint": 相对起飞点, "WGS84": EGM96 海拔 |
| waylineId | number | 航线 ID | - |
| autoFlightSpeed | number | 自动飞行速度 | 1-15 m/s |
| placemarks | array | 航点数组 | - |

### Placemark（航点）

| 字段名 | 类型 | 描述 | 取值范围 |
|--------|------|------|----------|
| index | number | 航点索引 | 从 0 开始 |
| executeHeight | number | 执行高度 | -1500 ~ 1500 米 |
| waypointSpeed | number | 航点速度 | 1-15 m/s |
| point | object | 坐标点 | - |
| waypointHeadingParam | object | 航向参数 | - |
| waypointTurnParam | object | 转弯参数 | - |
| actionGroups | array | 动作组列表 | - |
| isRisky | number | 是否危险点 | 0: 否, 1: 是 |

### Point（坐标点）

| 字段名 | 类型 | 描述 | 格式 |
|--------|------|------|------|
| coordinates | string | 坐标信息 | "经度,纬度,高度" |

### WaypointHeadingParam（航向参数）

| 字段名 | 类型 | 描述 | 取值范围 |
|--------|------|------|----------|
| waypointHeadingMode | string | 航向模式 | 见下方枚举 |
| waypointHeadingAngle | number | 航向角度 | -330 ~ 330 度 |

### waypointHeadingMode（航向模式枚举）

| 值 | 说明 |
|----|------|
| followWayline | 沿航线方向 |
| manually | 手动控制 |
| fixed | 固定角度 |
| smoothTransition | 平滑过渡 |
| towardPOI | 朝向兴趣点 |

### WaypointTurnParam（转弯参数）

| 字段名 | 类型 | 描述 | 取值范围 |
|--------|------|------|----------|
| waypointTurnMode | string | 转弯模式 | 见下方枚举 |

### waypointTurnMode（转弯模式枚举）

| 值 | 说明 |
|----|------|
| coordinateTurn | 协调转弯，不过点，提前转弯 |
| toPointAndStopWithDiscontinuityCurvature | 直线飞行，到点停 |
| toPointAndStopWithContinuityCurvature | 曲线飞行，到点停 |
| toPointAndPassWithContinuityCurvature | 协调转弯，过点转 |

### ActionGroup（动作组）

| 字段名 | 类型 | 描述 | 取值范围 |
|--------|------|------|----------|
| actionGroupId | number | 动作组 ID | 从 0 开始 |
| actionGroupStartIndex | number | 起始航点索引 | 从 0 开始 |
| actionGroupEndIndex | number | 结束航点索引 | 从 0 开始 |
| actionGroupMode | string | 执行模式 | "sequence": 串行, "parallel": 并行 |
| actionTrigger | object | 触发器 | - |
| actions | array | 动作列表 | - |

### ActionTrigger（动作触发器）

| 字段名 | 类型 | 描述 | 取值范围 |
|--------|------|------|----------|
| actionTriggerType | string | 触发类型 | 见下方枚举 |

### actionTriggerType（触发类型枚举）

| 值 | 说明 |
|----|------|
| reachPoint | 到达航点时执行 |
| betweenAdjacentPoints | 航段触发，均匀转云台 |
| multipleTiming | 等时触发 |
| multipleDistance | 等距触发 |
| betweenAdjacentPointsIncludeFirstPoint | 航段触发，均匀转云台（含首航点） |

### Action（动作）

| 字段名 | 类型 | 描述 |
|--------|------|------|
| actionId | number | 动作 ID |
| actionActuatorFunc | string | 动作类型 |
| actionActuatorFuncParam | object | 动作参数 |

动作类型和参数详见 [动作参数详细说明](/protocols/properties/actions.md)

## 使用示例

```typescript
import { WaylinesWpml } from "@gdu-proto-sdk/shared";

const waylines: WaylinesWpml = {
  version: "1.0.0",
  mission: {
    drone: {
      droneEnumValue: 67,
      droneSubEnumValue: 1,
    },
    payloads: [{
      payloadEnumValue: 53,
      payloadSubEnumValue: 1,
      payloadPositionIndex: 0,
    }],
    flyToWaylineMode: "safely",
    finishAction: "goHome",
    exitOnRCLost: "executeLostAction",
    executeRCLostAction: "goBack",
    takeOffSecurityHeight: 20,
    globalTransitionalSpeed: 15,
    autoReroute: {
      missionAutoRerouteMode: 0,
      transitionalAutoRerouteMode: 0,
    }
  },
  folders: [{
    templateId: 1,
    executeHeightMode: "relativeToStartPoint",
    waylineId: 1,
    autoFlightSpeed: 10,
    placemarks: [
      {
        index: 0,
        executeHeight: 100,
        waypointSpeed: 10,
        point: {
          coordinates: "113.9502,22.5431,100"
        },
        waypointHeadingParam: {
          waypointHeadingMode: "followWayline",
          waypointHeadingAngle: 0
        },
        waypointTurnParam: {
          waypointTurnMode: "coordinateTurn"
        },
        actionGroups: [{
          actionGroupId: 0,
          actionGroupStartIndex: 0,
          actionGroupEndIndex: 0,
          actionGroupMode: "sequence",
          actionTrigger: {
            actionTriggerType: "reachPoint"
          },
          actions: [{
            actionId: 0,
            actionActuatorFunc: "takePhoto",
            actionActuatorFuncParam: {
              takePhotoParam: {
                payloadPositionIndex: 0,
                payloadLensIndex: "0"
              }
            }
          }]
        }],
        isRisky: 0
      }
    ]
  }]
};
```
